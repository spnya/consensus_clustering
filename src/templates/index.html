Methods of consensus clustering and experimental comparisons

Welcome to the Consensus Clustering Experiment

This web application allows you to run consensus clustering experiments based on a given set of parameters. Consensus clustering is a method of combining multiple clustering results to produce a single consolidated clustering that is believed to be superior to the individual results.

You can configure the experiment by specifying the number of objects, clusters, ensemble size, mutation probability, and the type of shift transformation.
